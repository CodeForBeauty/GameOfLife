# Conway's Game of Life

Conway's Game of Life simulation made with C++ and OpenGL.

All of the simulation logic is run on CPU.

## How to run

This project is Visual Studio project, so it can be opened only in Visual Studio.

Upon application start, provide desired canvas width and height in the console.

The cell can be ticked on with left mouse button, and ticked off with right mouse button.

To resume or pause the simulation the P key can be pressed.
